About Me

I am recent graduate of the Master's program in the School of Library and Information Science at the University of Iowa. Along with that, I am familiar with working with datasets to clean, transform, and visualize the data within to provide insights. Some tools I have used before include Python and associated libraries, Power BI, Tableau, Excel, and Dash, a Python framework used to create data visualization dashboards from scratch using the Plotly visualization library.

I have enjoyed teaching myself these skills alongside completing my Master's program and think these have brought additional value to the academic experience. This drive to self-learn would be of value to either learn the responsibilities of a position or advance farther in a company.

Some of my other skills include HTML, CSS, and the Tailwind CSS framework, which I used to make this site.
